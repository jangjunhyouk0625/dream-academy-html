1 HTML - 구조 만들기

적용 후

<h1>안녕하세요</h1>

2 CSS - 디자인 꾸미기

안녕하세요

h1 {
  color: blue;
  text-align: center;
}

3 JavaScript - 동작 추가

적용 전

안녕하세요

적용 후

안녕하세요

<p id="after-text">안녕하세요</p>
<button onclick="changeText()">바꾸기</button>
<script>
  function changeText() {
    document.getElementById("after-text").innerText = 내용이 변경되었습니다;
  }
</script>
onclick은 클릭했을 때 실행되고, innerText는 글자를 바꿔줍니다.

JavaScript는 웹페이지에 동작을 추가하는 언어입니다.
