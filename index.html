2️⃣ CSS - 디자인 꾸미기

적용 전

안녕하세요!

적용 후

안녕하세요!

📜 코드 예시

h1 {
  color: blue;
  text-align: center;
}

button {
  background-color: green;
  color: white;
  padding: 10px;
  border-radius: 5px;
}

설명: CSS는 색상, 정렬, 여백 등을 조절해 **웹사이트를 꾸며주는 언어**입니다.

3️⃣ JavaScript - 동작 추가

적용 전

안녕하세요!

적용 후

안녕하세요!

📜 코드 예시

<p id="after-text">안녕하세요!</p>
<button onclick="changeText()">눌러보세요</button>

<script>
  function changeText() {
    document.getElementById("after-text").innerText = "✨ 바뀌었어요!";
  }
</script>

설명: JavaScript는 클릭, 팝업 등 **웹페이지에 동작을 추가하는 언어**입니다.
